# Forex Trading Simulator

## Project Description
This is a console-based Forex Trading Simulator developed using C++ and Object-Oriented Programming concepts.

The system simulates forex trading operations such as:
- Opening BUY and SELL trades
- Profit and loss calculation
- Account balance updates
- Trade history storage
- Market price simulation

---

## Features
- Account Management
- Currency Pair Management
- Trade Execution
- Profit/Loss Calculation
- File Handling
- Menu-driven Interface
